<h2>📌 Project Description</h2>
        Guess My Number is a simple browser-based number guessing game. 
        The player must guess a randomly generated number between 1 and 20. 
        The game provides feedback such as <strong>Too High</strong> or 
        <strong>Too Low</strong> and tracks the player's score and high score. 
        The game can be restarted at any time and also allows the high score to be reset.

     
   <h2>🛠 Technologies</h2>
      <ul>
        <li><strong>HTML5</strong></li>
        <li><strong>CSS3</strong></li>
        <li><strong>JavaScript (Vanilla JS)</strong></li>
        <li><strong>DOM Manipulation</strong></li>
      </ul>
    </section>

   
  <section>
      <h2>✨ Features</h2>
      <ul>
        <li>Random number generation between 1–20</li>
        <li>Player feedback messages (Too High, Too Low, Correct Number)</li>
        <li>Score system that decreases with each wrong guess</li>
        <li>High score tracking during gameplay</li>
        <li>Reset game functionality using the <strong>Again!</strong> button</li>
        <li>Ability to reset high score</li>
        <li>Visual feedback when the player wins (background color change)</li>
      </ul>
    </section>
